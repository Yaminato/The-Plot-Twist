# Happy New Year 2026 ï¿½

A romantic New Year's Eve webpage with animated hearts, scroll-triggered reveals, and Spotify Wrapped-style transitions.

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - All CSS styles
- `script.js` - JavaScript for animations and interactions
- `assets/` - Directory for media files
  - `music/` - Background music files
  - `photos/` - Personal photo files

## Features

- Countdown timer to New Year's
- Animated falling hearts
- Smooth scroll transitions between sections
- Progress bar animations
- **Split Layout Section**: Image on left, romantic text on right
- **Bento-Style Album**: Grid layout with different sized photo rectangles
- Personal letter section

## Customization

1. Update `HER_NAME` in `script.js`
2. Change `LETTER_TEXT` in `script.js`
3. Add your music file as `assets/music/happy_new_year.mp3`
4. Photos are already set up using your existing images:
   - `princess.jpg` (featured album photo)
   - `shy.jpg`, `peace.jpg`, `niceview.jpg` (memory photos)
5. **Countdown Date**: Set to January 1, 2026 (change in `script.js` if needed)
6. Update Spotify track: Replace the track ID in `index.html` (currently "Happy New Year" by ABBA)
7. Modify colors in `styles.css`

## Usage

Open `index.html` in a web browser. The page will show a countdown until New Year's, then unlock with animations.
